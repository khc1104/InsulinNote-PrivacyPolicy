# 인슐린노트(InsulinNote) 개인정보 처리방침

**인슐린노트(InsulinNote)** (이하 "본 앱")은(는) 사용자의 개인정보 및 건강 정보를 매우 중요하게 생각하며, 애플(Apple)의 App Store 심사 지침 및 관련 개인정보 보호 법령을 철저히 준수합니다. 

본 개인정보 처리방침은 본 앱이 사용자의 데이터를 어떻게 처리하는지 투명하게 안내하기 위해 작성되었습니다.

## 1. 개인정보 수집 및 이용
본 앱은 **사용자의 어떠한 개인정보나 민감한 건강 정보도 개발자나 제3자의 서버로 수집, 전송, 저장하지 않습니다.**

사용자가 앱 내에 입력하는 모든 데이터(사용하는 인슐린 종류, 인슐린 투여 기록 등)는 애플의 **SwiftData** 기술을 기반으로 전적으로 **사용자의 기기(Device) 내부에만 로컬로 저장**됩니다.

## 2. 데이터의 공유 및 제공
본 앱은 데이터를 외부 서버로 전송하지 않기 때문에, 사용자의 동의 없이 제3자(타 기업, 기관, 개인 등)에게 데이터를 제공하거나 공유하는 일은 원천적으로 불가능합니다.

## 3. 데이터의 보관 및 파기
앱 내의 모든 데이터는 오직 사용자의 기기에만 존재합니다. 
사용자가 기기에서 앱을 삭제하거나, 앱 내에서 데이터를 직접 삭제할 경우 해당 데이터는 기기에서 영구적으로 파기되며 복구할 수 없습니다.

## 4. 애플 건강(Apple Health) 앱 연동 (해당하는 경우)
본 앱이 Apple의 HealthKit API를 사용하여 건강 앱의 데이터를 읽거나 쓰는 경우, 해당 데이터 역시 기기 내에서만 처리되며 외부 서버로 전송되거나 마케팅, 광고 등의 목적으로 사용되지 않습니다. 사용자는 기기의 건강 앱 설정에서 언제든지 본 앱의 데이터 접근 권한을 관리할 수 있습니다.
*(※ HealthKit을 연동하지 않으셨다면 이 항목은 삭제하셔도 무방합니다.)*

## 5. 개인정보 처리방침의 변경
관련 법령 및 앱의 기능 변경(새로운 기능 추가 등)에 따라 본 개인정보 처리방침이 수정될 수 있습니다. 변경 사항이 있을 경우, 본 GitHub 페이지의 업데이트 내용 또는 앱 내 안내를 통해 사전에 공지할 것입니다.
